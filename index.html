1. print() Kullanımı

Kural: Tırnak içindeki yazı aynen yazılır. Tırnak yoksa işlem yapılır.

print("3+5")   # 3+5 (yazı)
print(3+5)     # 8   (işlem)

print("10-2")  # 10-2
print(10-2)    # 8

print("Merhaba")  # Merhaba
print(5 * 3)      # 15
print("5 * 3")    # 5 * 3
ÖNEMLİ: print("3+5") ekrana 3+5 yazar. print(3+5) ekrana 8 yazar. Tırnak varsa yazı, yoksa işlem!

2. if - elif - else Yapısı

puan = 75

if puan >= 85:
    print("Pekiyi")
elif puan >= 70:
    print("İyi")
elif puan >= 50:
    print("Geçtin")
else:
    print("Kaldın")
Kural: Önce en yüksek değer kontrol edilmeli! Aksi halde elif asla çalışmaz.

3. Hatalı Kod Bulma

Hata 1: Sıralama Hatası

puan = 75
if puan >= 50:         # 75 >= 50 Doğru, bu ÇALIŞIR
    print("Geçtin")
elif puan >= 70:       # Bu asla çalışmaz! İlk if doğruydu
    print("İyi")
Çözüm: Önce 85, sonra 70, sonra 50 kontrol edilmeli!

Hata 2: = ile == Karıştırmak

if sayi = 10:    # HATA! = atama yapar
    print("On")
if sayi == 10:   # DOĞRU! == karşılaştırma yapar
    print("On")

Hata 3: String ile Sayı Karşılaştırmak

yas = "18"       # Bu yazı (string)
if yas >= 18:    # HATA! yazı ile sayı karşılaştırılmaz
    print("Reşit")
yas = 18         # Sayı olmalı!
if yas >= 18:
    print("Reşit")

Hata 4: else'den Sonra Koşul Yazmak

if yas > 18:
    print("Reşit")
else yas == 18:    # HATA!
    print("Tam 18")
if yas > 18:
    print("Reşit")
elif yas == 18:
    print("Tam 18")
else:
    print("Reşit değil")

Hata 5: İki Nokta (:) Unutmak

if yas > 18     # HATA! : yok
    print("Reşit")
if yas > 18:
    print("Reşit")

4. 60 Paket İndirim Sorusu

SORU: 60 paket ve üzerine %20 indirim. Kaç paket alınırsa ne kadar ödenir?

birim_fiyat = 10
paket = int(input("Paket sayısı: "))

toplam = paket * birim_fiyat

if paket >= 60:
    indirim = toplam * 0.20
    odenecek = toplam - indirim
    print(f"İndirimli: {odenecek} TL")
else:
    print(f"İndirimsiz: {toplam} TL")

6. input() ile Veri Alma

ÖNEMLİ: input() her zaman yazı (string) alır. Sayı ile işlem yapacaksan int() veya float() eklemeyi UNUTMA!

Örnek 1: Ad ve Yaş

ad = input("Adınız: ")
yas = int(input("Yaşınız: "))   # int() ekledik!

print(f"Merhaba {ad}, {yas} yaşındasın.")

Örnek 2: Vize - Final Ortalaması

vize = float(input("Vize notu: "))
final = float(input("Final notu: "))

ortalama = vize * 0.40 + final * 0.60

if ortalama >= 50:
    print(f"Geçtin! Ortalaman: {ortalama}")
else:
    print(f"Kaldın! Ortalaman: {ortalama}")

7. Tek / Çift Bulma

sayi = int(input("Bir sayı girin: "))

if sayi % 2 == 0:
    print("Çift")
else:
    print("Tek")

% işareti kalanı (mod) bulur. Bir sayının 2'ye bölümünden kalan 0 ise çifttir.

8. Pozitif / Negatif / Sıfır

sayi = int(input("Bir sayı girin: "))

if sayi > 0:
    print("Pozitif")
elif sayi < 0:
    print("Negatif")
else:
    print("Sıfır")

9. İki Sayı Arasındakileri Listeleme

sayi1 = int(input("Birinci sayı: "))
sayi2 = int(input("İkinci sayı: "))

bas = min(sayi1, sayi2)
son = max(sayi1, sayi2)

for i in range(bas, son + 1):
    print(i)
Not: min() küçük, max() büyük sayıyı alır. +1 eklenir çünkü range son değeri dahil etmez.

Akış Şeması

┌─────────┐ │ BAŞLA │ └────┬────┘ │ ┌────▼────┐ │ sayi1 │ │ = oku │ └────┬────┘ │ ┌────▼────┐ │ sayi2 │ │ = oku │ └────┬────┘ │ ┌────▼────┐ │ bas=min │ │ son=max │ └────┬────┘ │ ┌────▼────┐ │ i=bas │ │ i<=son │───Hayır───┐ └────┬────┘ │ │ Evet │ ┌────▼────┐ │ │ yazdır │ │ │ (i) │ │ └────┬────┘ │ │ │ ┌────▼────┐ │ │ i=i+1 │ │ └────┬────┘ │ │ │ └────────┐ │ │ │ ┌────▼───────▼──┐ │ BİTİR │ └───────────────┘

Özet: Sık Yapılan Hatalar

  1. = ile == karıştırmak: if x = 5 (atama)  →  if x == 5 (karşılaştırma)
  2. int() eklemeyi unutmak: input() string alır  →  int(input()) sayı alır
  3. print() farkı: print("3+5") yazıdır  →  print(3+5) işlemdir
  4. if sıralaması: Önce en yüksek değer kontrol edilmeli
  5. İki nokta unutmak: if x > 5  →  if x > 5:
  6. else'ten sonra koşul yazılmaz: else:  ✅  else x > 5: ❌
  7. range() son değer: range(1, 5) = 1,2,3,4 (5 dahil değil!)
  8. % işareti: Kalan bulur. sayi % 2 == 0 ise çift sayıdır
